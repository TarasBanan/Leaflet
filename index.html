Многопользовательская игра через SSH. Никаких установок, никакого браузера — просто терминал и одна команда.

$ ssh player@85.239.36.136 нажми чтобы скопировать скопировано!

Как
подключиться

🪟 Windows
🐧 Linux
🍎 macOS
1
Открой PowerShell от имени администратора и установи OpenSSH:
Add-WindowsCapability -Online -Name OpenSSH.Client~~~~0.0.1.0 копировать
Альтернатива через winget: winget install Microsoft.OpenSSH.Beta
2
Открой любой терминал (PowerShell, Windows Terminal) и подключись:
ssh player@85.239.36.136 копировать
Рекомендуем Windows Terminal — он корректно отображает цвета
1
Установи OpenSSH клиент (если не установлен):
sudo apt install openssh-client копировать
2
Подключись:
ssh player@85.239.36.136 копировать
🍎 macOS — SSH уже встроен. Просто открой Terminal (Cmd + Space → "Terminal") и вводи команду.
1
Открой Terminal и подключись:
ssh player@85.239.36.136 копировать

Что
реализовано

🔐
Авторизация
Регистрация и логин в терминале. Пароли — SHA-256 хэш.
🏛️
Лобби
Автоподбор комнаты. Система пати — зови друзей по нику.
👑
Игра в реалтайме
Царь горы: физика толчков, баффы, шипы. До 4 игроков.
🛍️
Магазин скинов
Внутриигровая валюта. Символы и цвета персонажа.
🗺️
Конструктор уровней
Встроенный редактор карт прямо в игре.
🔑
Аккаунт админа
Управление комнатами и игроками.
