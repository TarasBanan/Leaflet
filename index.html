$ ssh player@85.239.36.136

Windows
PowerShell / cmd
1
Открой PowerShell от имени администратора и установи OpenSSH:
Add-WindowsCapability -Online -Name OpenSSH.Client~~~~0.0.1.0
или через winget: winget install Microsoft.OpenSSH.Beta
2
Подключись:
ssh player@85.239.36.136
🐧
Linux
apt / apt-get
1
Установи OpenSSH клиент:
sudo apt install openssh-client
2
Подключись:
ssh player@85.239.36.136
🍎
macOS — SSH уже встроен. Открой Terminal и сразу подключайся.
